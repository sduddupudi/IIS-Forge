# Support

Use GitHub Issues for:

- deployment failures
- missing dependency handling
- IIS path switching problems
- documentation gaps
- feature requests

When reporting a deployment problem, include:

- Windows version
- PowerShell version
- IIS version if known
- .NET target framework
- relevant profile JSON with secrets removed
- terminal output from the failed step

Do not include passwords, tokens, connection strings, certificates, or production secret files.
